# mini-guia-estudos-dio-01
Projeto criado para o primeiro desafio do "Bootcamp Bradesco - GenAI, Dados &amp; Cyber"

O objetivo desse projeto é criar um caderno temático no NotebookLM sobre algum assunto financeiro introdutório.
Para o meu projeto resolvi fazer algo em relação a uma dúvida que paira sobre muitas pessoas que tem vontade de iniciar no mundo dos investimentos, a escolha entre a renda fixa e a renda variável.

Escolhi algumas fontes que considerei confiáveis para que o usuário possa interagir com a IA e poder sanar as dúvidas de forma simples e ter, dessa forma, uma orientação sobre cada um dos assuntos.
